# Structure

- src (source code of the project)
- tests (all tests)
  - e2e
  - integration
  - unit
- docs (generated by npm run doc)
- coverage (generated by npm test)

# Commands
**npm run doc** generate docs
**npm start** run the code
**npm test** run tests and generate coverage report
